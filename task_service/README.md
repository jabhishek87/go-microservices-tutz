# Task Service


### Dev set up
```sh
# clone repo
git clone https://github.com/jabhishek87/go-microservices-tutz.git

# go task service dir
cd go-microservices-tutz/task_service

# install depedency
go mod tidy

# install air for Hot reload
go install github.com/air-verse/air@latest

# start Dev server
air
```
